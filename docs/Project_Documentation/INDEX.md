## 状态标记说明

- 【已完成】：代码与入口已存在，可直接运行或调用
- 【开发中】：代码存在但有缺口（如占位、缺少桥接、未完全接线）
- 【规划中】：配置/意图存在，但未形成完整可用链路
- 【无入口】：后端能力存在，但前端无入口页面/按钮
- 【未调用】：代码或接口定义存在，但当前仓库未被实际调用
- 【Demo/历史遗留】：存在展示或历史描述，不代表当前完整产品能力
