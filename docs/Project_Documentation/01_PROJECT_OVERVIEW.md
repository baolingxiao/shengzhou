# 01_PROJECT_OVERVIEW

## 一句话介绍项目

贾维斯是一个面向中文用户的本地优先 AI 助手产品，集成聊天、角色化记忆、语音交互、电脑代办和世界上下文同步能力于一体。

## 项目定位

- 定位：`AI 伴侣 + 生产力助理` 的混合产品
- 形态：`React PWA + FastAPI` 全栈应用
- 部署：支持本地开发、Docker、Linux 服务器（systemd + Nginx）
- 交付目标：既可个人使用，也可扩展为云端多用户服务

## 为什么存在

本项目要解决“通用 AI 不懂我、不能长期记住我、难以和真实生活联动”的问题，提供持续关系、上下文连续性和可执行行动。

## 解决什么问题

- 通用对话缺乏人格连续性：通过角色设定与信任度体系增强稳定体验
- 聊天历史碎片化：通过短/中/长期记忆宫殿与归档机制保持连续上下文
- 语音交互割裂：提供传统 STT/TTS 与 OpenAI Realtime 双链路
- 助理只会“说”不会“做”：提供 AI Agent 的待确认执行流程
- 生活上下文断层：提供 Shenzhou 世界引擎同步与主动触达能力

## 目标用户

- 个人高频 AI 用户（希望长期陪伴 + 高粘性对话）
- 需要“会记忆”的私人助理用户
- 需要语音入口的移动/桌面用户
- 对 AI Agent 电脑代办有需求的进阶用户
- 早期产品验证阶段的开发者/团队成员

## 产品价值

- 情绪与任务并重：既能聊，也能办事
- 长周期价值：越用越“懂你”
- 本地优先可控：核心数据在本地文件系统可追踪
- 可扩展架构：可接入不同模型、部署形态、外部世界引擎

## Elevator Pitch

“这不是一个只会回答问题的 AI，而是一个能持续记住你、理解你当下状态、并在必要时替你执行行动的私人智能系统。”
