# 11_INVESTOR_OVERVIEW

## 一句话介绍

贾维斯是一个“关系型 AI 助手”产品：把人格化对话、长期记忆、语音实时交互和可确认执行能力，整合到同一条用户连续体验链路中。

## 当前阶段判断

- 产品阶段：`0.1.0` 早期可用（MVP+）
- 形态：本地优先，全栈可部署
- 进展：核心能力闭环已形成，商业化与安全治理仍在补齐

## 目标市场（基于产品能力推断）

- 个人高频 AI 使用市场（陪伴 + 效率双需求）
- AI 个人助理市场（记忆与执行导向）
- 私有化/本地化部署需求场景（对数据可控敏感）

## 核心产品价值

- **持续关系价值**：不是“每轮重启”的聊天机器人
- **执行价值**：支持 Agent proposal-confirm-execute
- **上下文价值**：本地记忆宫殿 + 向量检索 + 外部生活上下文
- **交互价值**：文字、传统语音、Realtime 语音三通道统一

## 竞争优势（当前已体现）

- 记忆架构比普通聊天应用更深：文件可审计 + 向量可检索
- Agent 流程强调“确认门控”，兼顾能力与风险控制
- 语音链路双栈（传统/Realtime）适配不同场景
- 世界引擎同步能力可形成“现实事件驱动对话”差异化

## 技术壁垒

1. **记忆编排壁垒**：短中长期联动 + 动态召回门控
2. **人格演进壁垒**：信任度体系与工作模式影响行为策略
3. **执行安全壁垒**：pending 持久化 + gate 风险控制 + 用户确认
4. **集成壁垒**：聊天、语音、Agent、外部世界数据在同一系统内闭环

## AI 能力拆解

- LLM 路由：general/fast/deep/code 多路由
- Provider：Doubao / Claude（主对话），OpenAI（Realtime）
- 合规层：回复审查与重写链路
- 追踪层：trace_id 贯穿前后端

## 商业模式（可落地方向）

### 短期（可验证）

- Pro 订阅：更高配模型、更多记忆容量、更多语音额度
- 私有部署服务：面向团队或高隐私客户

### 中期（架构可承接）

- Agent 自动化套餐（任务执行额度）
- 行业模板与角色模板市场

### 长期（需产品化完善）

- B2B SaaS 控制台（租户、权限、审计、策略）
- 开放插件与生态分成

## 主要风险与当前缺口

- 安全：明文密码存储、部分系统接口鉴权不足
- 产品化：Shenzhou 能力前端入口不足
- 模块完整度：Topic Radar/Reminder 仍处于开发中状态
- 发布治理：缺少 tag 与规范化 release 流程

## 12个月里程碑建议

- **0-3个月**：安全基线与入口补齐（鉴权、密码哈希、Shenzhou 面板）
- **3-6个月**：Topic Radar/Reminder 完整化，形成“主动智能”差异化
- **6-12个月**：SaaS 化控制台 + 计费能力 + 团队版试点

## 投资结论（基于代码现状）

这是一个“技术深度明显高于普通聊天应用”的早期产品，核心引擎已成型，当前价值在于尽快把工程能力转化为可规模化交付的产品体验与商业闭环。
