# 04_USER_GUIDE

> 面向第一次接触本项目的普通用户。  
> 目标：按照步骤操作即可跑起来，不需要先懂代码。

## 0. 你将得到什么

- 能登录并开始聊天
- 能用语音输入或语音对话
- 能创建自己的 AI 人物（普通用户）
- 能在开发者账号下管理记忆和系统权限

---

## 1. 安装（本地电脑）

### 1.1 准备环境

请先安装：

- Python 3.10+
- Node.js 20+
- npm

### 1.2 下载并进入项目目录

如果你已经拿到项目文件，直接打开终端进入项目目录即可。

### 1.3 初始化配置

复制环境变量模板：

```bash
cp .env.example .env
```

然后编辑 `.env`，至少确认这些项：

- `JARVIS_AUTH_USER` / `JARVIS_AUTH_PASSWORD`
- 模型密钥（例如 `DOUBAO_API_KEY`）
- 语音需要时再配置 `ELEVENLABS_API_KEY`、`ELEVENLABS_VOICE_ID`

---

## 2. 启动

### 2.1 一键启动（推荐）

```bash
npm run start
```

启动后默认地址：

- 前端：`http://127.0.0.1:5190`
- 后端：`http://127.0.0.1:8766`

### 2.2 停止服务

```bash
npm run stop
```

---

## 3. 注册与登录

打开网页后会先看到登录页。

你可以：

- 直接登录已有账号
- 点击“去注册”，注册新账号后自动登录

登录成功后会进入启动页，再点击“来了，老弟！”进入主界面。

---

## 4. 第一次创建你的 AI（普通用户）

如果你是普通用户（`role=user`），首次登录会看到“创建你的 AI 人物”页面：

需要填写：

- 角色名字
- 回复风格 prompt
- 至少一个模型 API Key（四选一即可）

保存后才能进入聊天。

---

## 5. 开始聊天

进入主界面后：

1. 在输入框输入文字
2. 按回车或点击“发送”
3. 等待 AI 回复

状态提示说明：

- `就绪`：可发送消息
- `对方正在输入......`：模型处理中
- `服务离线`：后端未启动或不可达

---

## 6. 语音功能

## 6.1 语音输入（说完后手动发送）

- 点击输入框右侧麦克风按钮
- 说完后再次点击，文字会自动填入输入框
- 再按回车或发送按钮

## 6.2 语音对话（自动听说）

- 点击“语音对话：开”（或 Realtime 语音开关）
- 系统会持续听你说话并回复
- 可随时关闭

## 6.3 文字朗读

- 点击“文字朗读：开”
- AI 文本回复会自动播放语音

---

## 7. 记忆管理（开发者账号）

如果你使用开发者账号，会看到右下角“记忆”按钮。

可执行：

- 查看短期/中期/长期记忆
- 标记重要记忆
- 删除记忆
- 触发日结/周结/月结维护

普通用户默认没有这个入口。

---

## 8. 系统权限（macOS）

如果你在 macOS 使用 Agent 功能：

- 点击左下角“权限”按钮
- 查看辅助功能、屏幕录制是否已授权
- 可点击“一键授权”或跳转系统设置

---

## 9. 更新与安装到桌面

PWA 渠道会自动提示：

- 安装到桌面
- 前端 Service Worker 更新
- 后端 Git 更新（若开启）

按弹窗引导操作即可。

---

## 10. 常见故障快速处理

- 页面提示服务离线：确认 `npm run start` 已成功
- 登录失败：检查 `.env` 的账号密码是否一致
- 语音不可用：检查麦克风权限和语音相关密钥
- Realtime 不可用：检查 `OPENAI_API_KEY` 和 `VITE_ENABLE_REALTIME_VOICE`
- 回复异常慢：检查模型 API 可用性与网络
