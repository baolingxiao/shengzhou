# Vibe Motion Prompt

Use this when iterating on the Neural Pal wakeup feel:

---

Design a warm, cinematic AI awakening — not sci-fi, not cyberpunk.

**Mood:** Soft morning light on cream paper. A presence waking gently, like someone opening their eyes.

**Motion language:**
- Slow breathing (scale ±2%, opacity ±25%)
- Collapse inward with grace — no snap, no rubber band
- Morph should feel like water finding its shape
- Text arrives like a whisper, not a notification
- Interface emerges from calm, not a dashboard boot sequence

**Color:** Warm off-white `#F8F3EA`, orange-white glow `#FFD4A8`, dark warm gray text `#3D3630`

**Avoid:** Neon blue, grid overlays, scan lines, glitch, HUD corners, typing sounds, matrix rain

**Reference feeling:** The intimacy of *Her* — original execution only.

**Easing:** `[0.22, 1, 0.36, 1]` everywhere important
